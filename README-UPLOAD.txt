KAVI PAST PAPERS — UPLOAD GUIDE

1. Extract this ZIP file.
2. Open your GitHub repository: kavi-past-papers
3. Choose Add file > Upload files.
4. Upload ALL files and the assets folder.
5. Choose Commit changes.
6. Wait 1–3 minutes and refresh your GitHub Pages link.

IMPORTANT
- This package replaces the homepage and category pages with the new complete design.
- Dark mode works automatically and remembers the visitor's choice.
- Years 2029 to 2023 are included everywhere.
- To add PDF links, edit papers-data.js using the example shown at the top of that file.
- Replace the placeholder contact email before publishing.
- Ad boxes are placeholders only. Add your real AdSense code after Google approves your site.
